# CTA Architecture

Use this skill when the funnel needs a clearer next step.

## Process

1. Map the current entry point, CTA, and destination step.
2. Identify where intent is lost through vague language or extra friction.
3. Pair each CTA with the stage of buyer readiness it serves.
4. Recommend one primary CTA and one fallback CTA only.

## Output

- Funnel step map
- Primary CTA
- Fallback CTA
- Conversion friction
