# Offer Design

Use this skill when the task is to improve packaging or value clarity.

## Process

1. Restate the ICP pain and desired outcome.
2. Describe the offer promise, mechanism, and expected proof points.
3. Identify friction created by weak packaging or unclear scope.
4. Rewrite the offer as something sales and marketing can both use.

## Output

- Offer statement
- Value components
- Proof requirements
- Friction notes
