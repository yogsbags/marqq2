# Dev Skill

Use the measurement and experimentation playbooks aligned to Corey Haines' marketing skills.

## Primary Skill Families
- `analytics-tracking`: event design, source attribution, KPI definitions, tracking audits
- `ab-test-setup`: experiment design, hypothesis framing, variant measurement, decision thresholds
- `onboarding-cro`: activation and time-to-value analysis when instrumentation reveals post-signup drop-off
- `paid-ads`: campaign metric interpretation when channel data affects spend allocation
- `page-cro`: page-level conversion analysis when analytics point to landing-page leakage

## Execution Rules
- Start with measurement validity before interpreting performance.
- Compare observed performance against a baseline, prior period, or control.
- Separate instrumentation issues from actual business performance changes.
- Recommend experiments only when the measurement path is credible enough to support decisions.
