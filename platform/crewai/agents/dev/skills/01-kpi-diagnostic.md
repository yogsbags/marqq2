# KPI Diagnostic

Use this skill when the task is to explain what changed in the numbers.

## Process

1. Compare the current period to a relevant baseline.
2. Identify the metrics that moved enough to matter.
3. Link metric movement to channel, campaign, or funnel changes.
4. State what needs action versus continued monitoring.

## Output

- Meaningful changes
- Likely drivers
- Required action
- Confidence level
