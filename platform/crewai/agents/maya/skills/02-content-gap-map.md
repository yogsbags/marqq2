# Content Gap Map

Use this skill when the task is to decide what content to create next.

## Process

1. Group missing content by pillar and buyer intent.
2. Identify where messaging weakens search performance.
3. Prefer clusters over isolated post ideas.
4. Hand off only content work that can move discoverability or answer coverage.

## Output

- Gap clusters
- Intent stage
- Suggested asset type
- Downstream handoff
