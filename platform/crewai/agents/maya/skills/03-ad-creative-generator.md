# Ad Creative Generator

Use this skill when the task is to write ad copy for a specific platform, audience, and offer.

Maya generates platform-native creatives — not generic copy pasted across platforms. Character limits, tone, and format differ by platform.

## Platform Limits

| Platform | Headline | Primary Text | Description |
|----------|----------|-------------|-------------|
| Meta (Facebook/Instagram) | 40 chars | 125 chars | 30 chars |
| Google Search | 30 chars (3 headlines) | 90 chars (2 descriptions) | — |
| LinkedIn | 150 chars | 600 chars | 70 chars |

## Angles to Test

Always generate at least one ad per angle. All 5 angles per campaign when volume allows:

1. **Pain-focused** — name the specific frustration the ICP lives with
2. **Outcome-focused** — lead with the transformation or result, skip the product
3. **Social proof** — a real or plausible customer result with numbers
4. **FOMO** — scarcity, deadline, or "everyone else is already doing this"
5. **Curiosity** — open loop that makes the viewer need to click to resolve it

## Process

1. Identify the platform, the ICP segment, and the offer.
2. Write one ad per angle. Label each with platform, angle, and sequence number.
3. Stay within character limits — count carefully. Do not approximate.
4. Write the visual brief for each ad (what the designer/AI image tool should create).
5. Assign a CTA button appropriate to the funnel stage: awareness = Learn More, consideration = Book Demo / See How It Works, conversion = Get Started / Start Free Trial.
6. For Indian audiences: use ₹ instead of $, reference local pain points, and include a WhatsApp CTA note for SMB-targeted Meta ads.

## Output

Use this structure per ad:

---

**Ad [N] — [Platform] — [Angle]**
Headline: [within character limit]
Primary text: [within character limit]
Description: [within character limit — skip if not applicable]
Visual brief: [2–3 sentences describing what the image or video should show, including composition, mood, and any text overlay]
CTA button: [exact button label]
Target segment: [which audience this is written for]

---

[Repeat for each ad]

## India-Specific Creative Notes

- Use ₹ for any pricing references when targeting Indian audiences.
- Reference familiar pain points: "without hiring a full team", "for lean Indian startups", "SEBI-compliant", "GST-ready".
- For Meta ads targeting India SMB: add a note "Consider WhatsApp CTA (Send Message) over website clicks — lower friction for this segment."
- Avoid overly formal British English. Indian B2B buyers respond to direct, practical language.
- LinkedIn India: finance, SaaS, and HR audiences are highly active. Use seniority signals in copy ("For Founders", "For CHROs at 200+ headcount").

## Quality Checks

Before delivering creatives:
- All character limits respected
- No two ads use the same opening word
- Every ad has exactly one CTA
- Visual brief is specific enough for an image tool or designer to act on without a call
