# Segment Research

Use this skill when the task is to sharpen ICP detail from partial evidence.

## Process

1. Start from current MKG positioning and ICP fields.
2. Infer segment traits only when multiple signals point the same way.
3. Capture buyer pain, urgency, and likely purchase blockers.
4. Mark assumptions clearly so Neel can decide what is strategy versus fact.

## Output

- ICP segments with confidence
- Buyer pain points
- Trigger events
- Research gaps
