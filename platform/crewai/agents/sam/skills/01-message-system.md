# Message System

Use this skill when the task is to sharpen the core language of a campaign.

## Process

1. Identify the audience, pain, promise, and proof.
2. Build one clear message spine that can survive across channels.
3. Remove unclear jargon or duplicate claims.
4. Keep language tied to the actual offer and funnel step.

## Output

- Core message
- Supporting proof
- CTA language
- Objection handling
