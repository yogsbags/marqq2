# Email Sequence Writer

Use this skill when the task is to write a series of emails for a specific lifecycle trigger or campaign goal.

## Sequence Types Sam Can Write

- **Nurture** — warm an MQL over 2–4 weeks with value before asking for a demo
- **Onboarding** — guide a new user through activation in the first 7–14 days
- **Cold outreach** — 3–5 touch sequence for a named ICP with no prior relationship
- **Win-back** — re-engage churned or lapsed users/leads
- **Trial conversion** — convert trial users to paid before expiry

## Process

1. Identify the sequence type and the triggering event (signup, inbound lead, trial start, etc.).
2. Decide the number of emails needed. Default: nurture = 4, onboarding = 5, cold = 3, win-back = 3, trial = 4.
3. For each email: write subject, preview text, body (150–300 words), one CTA, and send timing.
4. Vary the angle across the sequence — do not repeat the same hook. Use: value delivery → social proof → objection handling → urgency.
5. Keep body copy scannable: short paragraphs, max 3 sentences each, no walls of text.
6. Every email has exactly one CTA. No secondary links unless the CTA is a P.S. suggestion.

## Output

Use this structure for each email in the sequence:

---

**Email [N] of [Total]: [Subject Line]**
Preview text: [under 90 characters — completes or contrasts the subject line]

Body:
[Full email body — 150–300 words. Write in plain conversational English. Use the recipient's role and pain point from the ICP. Avoid filler phrases like "I hope this email finds you well." Open with the problem or insight, not with a self-introduction.]

CTA: [Button text — under 5 words] → [What happens when clicked]
Send timing: Day [X] after [trigger event]

---

[Repeat for each email]

## Quality Checks

Before finishing the sequence:
- Subject lines: no more than 2 using questions; no subject line repeats the same format twice in a row
- Preview text: never starts with "In this email..."
- CTAs: all different — no two emails share the same button text
- Tone: matches the sequence type (warmer for nurture, crisper for cold, urgent for trial)
