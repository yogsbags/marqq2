# Nurture Copy

Use this skill when the task is to improve follow-up messages.

## Process

1. Match the message to the recipient's stage of readiness.
2. Keep the copy focused on one action.
3. Use proof, specificity, and clarity over cleverness.
4. Make the next step easy to understand and low-friction.

## Output

- Subject or opener
- Body message
- CTA
- Follow-up angle
