# Content Brief

Use this skill when the task is to create a publishable brief quickly.

## Process

1. Name the audience, offer, and content goal.
2. Define the core angle and the proof needed in the piece.
3. Outline the structure with clear CTA placement.
4. Keep the brief specific enough for rapid production.

## Output

- Title or asset name
- Audience and goal
- Outline
- CTA
