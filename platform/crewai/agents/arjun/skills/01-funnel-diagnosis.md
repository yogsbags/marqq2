# Funnel Diagnosis

Use this skill when conversion performance is weak or unclear.

## Process

1. Map the current funnel stages and expected movement between them.
2. Identify where drop-off is highest.
3. Tie funnel weakness back to message, offer, or routing issues.
4. Recommend the smallest fix with the clearest conversion impact.

## Output

- Weakest funnel stage
- Likely cause
- Recommended fix
- Expected impact
