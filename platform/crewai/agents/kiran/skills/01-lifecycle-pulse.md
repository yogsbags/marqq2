# Lifecycle Pulse

Use this skill when the task is to assess retention or nurture health.

## Process

1. Review current audience touchpoints after acquisition.
2. Find where engagement weakens across time or sequence.
3. Tie drops to content, timing, or message mismatch.
4. Recommend the next nurture move with the best chance of lift.

## Output

- Lifecycle stage
- Engagement issue
- Recommended nurture move
- Follow-up metric
