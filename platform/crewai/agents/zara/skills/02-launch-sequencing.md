# Launch Sequencing

Use this skill when multiple campaigns or assets need an order of operations.

## Process

1. Identify the asset or offer that should lead.
2. Define which supporting assets must exist before launch.
3. Stage the rollout so feedback from earlier moves informs later ones.
4. Note dependencies for Riya, Kiran, Sam, and Arjun.

## Output

- Launch order
- Required dependencies
- Success signals
- Handoff list
